# freyr-gui

A cross-platform front-end GUI for freyr-js written in Rust

## Setup

• Install `pyqt5`

• Clone and preview

  ``` bash
  # Clone this repo
  git clone https://github.com/miraclx/freyr-gui
  cd freyr-gui
  # Run the preview
  pyuic5 -p main.ui
  ```
